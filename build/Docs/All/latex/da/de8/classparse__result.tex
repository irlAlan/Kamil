\doxysubsection{Detailed Description}
The result of a parsing operation. 

\textbackslash{}availability {\bfseries{This type only exists when exceptions are disabled.}} Otherwise \mbox{\hyperlink{classparse__result}{parse\+\_\+result}} is just an alias for toml\+::table\+: \textbackslash{}cpp \#if TOML\+\_\+\+EXCEPTIONS using \mbox{\hyperlink{classparse__result}{parse\+\_\+result}} = table; \#else class \mbox{\hyperlink{classparse__result}{parse\+\_\+result}} \{ // ... \#endif \textbackslash{}ecpp

\textbackslash{}detail A \mbox{\hyperlink{classparse__result}{parse\+\_\+result}} is effectively a discriminated union containing either a toml\+::table or a toml\+::parse\+\_\+error. Most member functions assume a particular one of these two states, and calling them when in the wrong state will cause errors (e.\+g. attempting to access the error object when parsing was successful). \textbackslash{}cpp toml\+::parse\+\_\+result result = toml\+::parse\+\_\+file(\char`\"{}config.\+toml\char`\"{}); if (result) do\+\_\+stuff\+\_\+with\+\_\+a\+\_\+table(result); //implicitly converts to table\& else std\+::cerr \texorpdfstring{$<$}{<}\texorpdfstring{$<$}{<} \char`\"{}\+Parse failed\+:\textbackslash{}n\char`\"{}sv \texorpdfstring{$<$}{<}\texorpdfstring{$<$}{<} result.\+error() \texorpdfstring{$<$}{<}\texorpdfstring{$<$}{<} \char`\"{}\textbackslash{}n\char`\"{}; \textbackslash{}ecpp
