If a TOML document contains lots of codepoints outside of the ASCII range you may find that your source\+\_\+positions don\textquotesingle{}t match those given by a text editor (typically the line numbers will be accurate but column numbers will be too high). {\bfseries{This is not an error.}} I\textquotesingle{}ve chosen this behaviour as a deliberate trade-\/off between parser complexity and correctness.
