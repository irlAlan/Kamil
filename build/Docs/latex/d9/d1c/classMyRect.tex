\doxysubsection{Detailed Description}
gives extra functionality to Float\+Rect 

Uses Float\+Rect for the ability to collision detect better than Rectangle\+Shape and inherits from Drawable so we are able to keep uniform sytax of window.\+draw(\+Drawable object)
