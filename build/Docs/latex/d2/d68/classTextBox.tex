\doxysubsection{Detailed Description}
A class that makes a Textbox in SFML. 

The class creates a textbox for inputting and handling text and \mbox{\hyperlink{classKeyboard}{Keyboard}} commands and allows the use of commands in the secondary textbox cmdbox
